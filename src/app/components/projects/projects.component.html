Cross-Platform-Apps mit Ionic + Angular

Store Apps

16+ Apps (iOS / Android) für US-amerikanische Tankstellen- und Convenience Store Ketten. Erstellt mit dem Ionic Framework zur Cross-Plattform-Entwicklung mit Angular als zugrunde liegendes JS-Framework.


Features u.a.:
  • Bezahlen per App an der Zapfsäule (Pay At Pump)
  • Location- / Store-Manager inkl. Benzinpreisanzeige
  • Self-Check-Out (GoSkip)
  • Push Benachrichtigungen, Barcode-Scanner
  • Benutzerverwaltung
  • Bereitstellen aktueller Angebote / Coupons.
  • Einbinden unserer Order Ahead Web App (s.o.)
  • Anbindung an Zahlungs- und Kassensysteme via APIs/SDKs (z.B. Zipline/PDI, CGo, Stripe).
  • Anbindung an Loyaltysysteme im Conexxus/NCR RLM Standard (APIs, Multiplexer).
  • Anbindung an das Altria (vormals Philip Morris) Tobacco Loyalty + Coupons Programm via API.
  • Kommunikation zwischen Endkunden + Kundenservice.
  • Eigene Backend-Lösung (s.u.)

Loyalty-Server / System zur Anbindung an Kassensysteme, Zapfsäulen + Store Apps

Loyalty-Server

Loyalty-System im Conexxus und NCR RLM Standard als Node.js / Express Server.
Features u.a.:

  • Einbindung / Umsetzung beliebiger Loyaltyprogramme (Rabatte, Coupons, Punkte, Staffelungen).
  • Anbindung an Kassensysteme und Tankstellenzapfsäulen (Conexxus, NCR RLM).
  • Anbindung an Web- und Store Apps.
  • Unterstützung virtueller + physischer Loyaltykarten.
  • Datenverwaltung
  • Diagnosetools, Ereignisprotokollierung
  • Backend für Loyaltykunden (Support)

Tech-Stack u.a.: Node.js, Express, TypeScript (ts-node), XML-Body-Parser, Sentry, Loggly, AWS EC2
